## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Uses json-server

npm install -D json-server
uses tanstack/db.json
npx json-server --watch db.json --port 3001
